Arkiv Ethereum Examples

Demo applications showcasing what you can build with Arkiv, a universal and cost-efficient data layer for Ethereum.

About

This repository contains two example implementations that demonstrate how to query and visualize Ethereum network data using Arkiv. Both examples show real-time gas prices, block information, and transaction analytics.

Examples

  • Showcase Dashboard →
    React + Vite + TypeScript

    Modern dashboard with interactive charts, heatmaps, and real-time data visualization. Features a polished UI built with React and shadcn/ui components.

  • Core Dashboard →
    Vanilla JavaScript + HTML + CSS

    Lightweight implementation with zero framework dependencies. Demonstrates the same functionality with minimal overhead and a clean, straightforward approach.
